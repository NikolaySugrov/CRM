# CRM

Учебный CRM-интерфейс для работы с клиентами. Проект сделан на React, TypeScript и Vite.
Демо https://crm-hxje.vercel.app/
## Возможности

- добавление клиентов;
- удаление клиентов;
- изменение статуса клиента;
- поиск по имени и компании;
- фильтрация по статусу;
- статистика по клиентам и бюджету;
- сохранение данных в localStorage;
- базовая валидация формы;
- адаптированная UI-структура: sidebar, таблица, форма и блок статистики.

## Стек

- React
- TypeScript
- Vite
- CSS
- lucide-react
- localStorage

## Запуск проекта

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
